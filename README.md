# 박인서 (In-seo Park)

**20년 축구선수에서 개발자로. 문제를 만나면 원인을 찾을 때까지 파고듭니다.**

8살부터 축구만 하다 은퇴 후 개발에 뛰어들었습니다.
낯선 분야였지만 선수 시절 훈련하듯 매일 학습해, 실제 서비스를 만들고 배포하며 성장하고 있습니다.

- 실사용 SaaS 직접 개발·운영 (기획 → 개발 → 배포 → 운영)
- AI 출력 오류율 80% 이상 개선 경험 (오류 유형 분류 → 프롬프트 재설계 + 검증 로직)
- 자격증: 프로그래밍기능사, 리눅스마스터 2급, 컴퓨터활용능력 2급

## Tech Stack

**Frontend** React · TypeScript · JavaScript · HTML/CSS
**Backend** Python · Django · Node.js · Firebase · Supabase
**AI/API** Claude API · Gemini API · OpenAI API · 공공데이터 API
**Deploy** Vercel · Netlify · Firebase Hosting
